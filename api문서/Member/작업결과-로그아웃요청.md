로그아웃 요청 API (/api/member/logout)  
■ request
   - method: post
  
■ response:  
   - 200: 성공  
      - case1. return : logout  
  
■ 실제동작코드  
MemberController.java  
```java
@PostMapping("/logout")
public ResponseEntity<JSONResult> logout() {
  return ResponseEntity.status(HttpStatus.OK).body(JSONResult.success("logout"));
}
```
